# Instalation guide
## 1. Install GNU Backgammon
Download GNU Backgammon at https://www.gnu.org/software/gnubg/#TOCdownloading

Install it.

## 2. Install PyJulia
```sh
py -m pip install julia
```

## 3. Install PyCall.jl
Dans la console Python
```py
import julia
julia.install()
```
